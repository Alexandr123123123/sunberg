# Техническая спецификация: Страница услуг (Services)

Страница услуг является ключевой точкой конверсии проекта Sunberg, демонстрируя инженерную экспертизу и комплексный подход к солнечной энергетике.

## 1. Структура страницы
Страница разделена на следующие функциональные блоки (виджеты):

1.  **Hero (SpHero)**: Приветственный блок с четким ценностным предложением.
2.  **Sectors (SpSectors)**: Отраслевая специализация (Residential, Commercial, Industrial).
3.  **TechBanner (SpTechBanner)**: Технологический акцент на TOPCon-эффективности.
4.  **Solutions (SpSolutions)**: Сетка специализированных решений с встроенным CTA.
5.  **DeepDive (SpDeepDive)**: Каскадный блок, детально описывающий этапы инженерии через систему табов.
6.  **Hardware (SpHardware)**: Блок доверия с логотипами технологических партнеров.
7.  **CaseStudy (SpCaseStudy)**: Демонстрация реального примера реализации (Villa Eriksson).
8.  **FAQ (Faq)**: Ответы на частые вопросы клиентов.
9.  **BottomCta (SpBottomCta)**: Финальный призыв к действию перед футером.

## 2. Функциональные требования
- **Адаптивность**: Полная поддержка устройств от мобильных (320px) до ультрашироких мониторов.
- **Интерактивность**: Плавное переключение табов в DeepDive без перезагрузки страницы.
- **Анимации**: Использование `viewport`-анимаций (появление при скролле) для всех крупных блоков.
- **SEO**: Использование семантических тегов (section, h1-h3) и оптимизированных изображений.

## 3. Дизайн-система (Nordic Soft Corporate)
- **Цвета**: Преобладание мягких природных оттенков, акцентный зеленый (#5B7E5F).
- **Типографика**: Использование шрифтов Inter и Outfit для заголовков.
- **Эстетика**: Асимметричные сетки, большие отступы (whitespace), эффект Glassmorphism для карточек.
