# ☀️ Sunberg Solar Platform

> **Premium engineering solutions for a sustainable future.**
> This platform is a high-fidelity digital ecosystem built for a leading solar energy firm, following the **Feature-Sliced Design (FSD)** methodology.

---

## 🏛️ Project Architecture
The platform is built on a robust, modular foundation that ensures scalability and visual excellence.

- **Frontend**: React 18+ with Vite
- **Styling**: CSS Modules (Scoped Styles)
- **Animation**: Framer Motion (Architectural Rhythm)
- **Architecture**: [**Feature-Sliced Design (FSD)**](./architecture.md)

## 🚀 Quick Start

### 1. Prerequisites
- Node.js (v18 or higher)
- NPM or Yarn

### 2. Installation
```bash
npm install
```

### 3. Development
```bash
npm run dev
```

## 🎨 Design Philosophy: "Nordic Soft Corporate"
Sunberg follows a unique design language that balances technical precision with organic, premium aesthetics:
- **Whitespace**: Generous breathing room for content.
- **Typography**: Precision headers using *Outfit* and *Inter*.
- **Motion**: Subtle, high-end transitions and parallax effects.
- **Authenticity**: Custom-generated AI assets for a unique visual identity.
